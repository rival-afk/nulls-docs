# Как начать создавать моды для Null's Brawl?

### Присоединитесь к нашему чату и установите специальный клиент

На данный момент, все важные обсуждения касаемо модификаций и их разработки происходят в телеграм-чате: [t.me/nb_mods](https://t.me/nb_mods).

Там же, в разделе #Announcements, вы сможете скачать специальный APK. Установите его, чтобы разблокировать следующий шаг.

### Создайте папку с модами

На внешнем хранилище создайте пустую папку с любым названием. Например, NBMods.

Откройте Null's Brawl, зайдите в настройки загрузчика модификаций и выберите эту папку в качестве внешнего источника.

### Создайте свой первый мод

У каждого нового мода должен быть свой уникальный ID в формате UUID. Перейдите на сайт [uuidgenerator.net](https://uuidgenerator.net) чтобы сгенерировать его.

Перейдите в папку с модами, которую вы создали на предыдущем шаге. Внутри нее создайте еще одну папку, название которой будет полностью совпадать со сгенерированным UUID мода.
Теперь эта папка — папка вашего мода.

Внутри нее создайте файл с названием **content.json** (маленькими буквами — это важно). Внутри этого файла поместите следующий код:

```json
{
  "@title": "Мой первый мод",
  "@description": "И его описание",
  "@gv": 65,
  "ru": {
    "TID_JESTER": {
      "RU": "КЛОУН"
    }
  }
}
```

<details>
<summary>
Как должна выглядеть структура файлов и папок
</summary>

```
╔ Внешнее хранилище
╚╦ 📁 NB Mods
 ╚╦ 📁 bd850738-67cb-4583-92c4-a12f4f6080d1
  ╚═ 📝 content.json
```

Но это лишь пример. Папки могут назваться по-другому.
</details>

### Включите свой первый мод и проверьте его

Откройте Null's Brawl, снова зайдите в настройки загрузчика модификаций и там вы должны увидеть свою модификацию в списке модов.

Включите её и вернитесь в игру. Если всё сделано правильно, то теперь Честер будет называться "КЛОУН".

### Где взять актуальные файлы таблиц?

Файлы таблиц игры (CSV) не хранятся в этом репозитории, так как они часто обновляются. Чтобы получить актуальную версию:

1. **Telegram-канал hvh:** в канале https://t.me/hvh периодически публикуются архивы с файлами таблиц для текущей версии игры.
2. **Извлечение из APK:** вы можете извлечь файлы таблиц напрямую из APK-файла игры с помощью архиватора (они находятся в папке `assets/csv`).

**Важно:** не стоит коммитить CSV-файлы в репозитории документации — они устаревают при каждом обновлении баланса или выходе новой версии.

### Дальнейшие шаги

Чтобы создавать более сложные моды, вам стоит:

1. Узнать как устроены файлы игры;
2. Изучить подробнее [формат модификаций](Format.md).
