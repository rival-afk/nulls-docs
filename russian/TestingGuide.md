## Дополнительный гайд 6: Тестирование модов без подписи

Чтобы тестировать моды в процессе разработки, не обязательно каждый раз подписывать их. Null's Brawl предоставляет специальный клиент и возможность загружать моды из папок.

### Способ 1: Специальный APK для разработчиков

Начиная с версии 65.165 существует отдельный APK, который позволяет загружать неподписанные моды (прямо из ZIP или JSON).

**Где скачать:**  
Ссылка на актуальную версию появляется в анонсах (например: https://tempweb.nullsusercontent.com/fpapk/nb_65.165_release_412b740e.apk?allowUnsignedMods=1).

**Важно:** этот APK предназначен **только для разработчиков**. Используйте его для тестирования своих модов. Запрещено устанавливать чужие неподписанные моды. Нарушение правил может привести к блокировке аккаунта.

### Способ 2: Ручная установка через папки (для любого клиента)

Этот метод работает и в обычном Null's Brawl, но требует, чтобы мод был распакован в определённую папку.

**Пошаговая инструкция:**
1. **Создайте папку для модов**  
   Во внутренней памяти устройства создайте папку, например, `NullsBrawlMods`.

2. **Сгенерируйте UUID** для вашего мода на https://uuidgenerator.net. Скопируйте его.

3. **Создайте подпапку с именем UUID**  
   Внутри `NullsBrawlMods` создайте папку, название которой точно совпадает с полученным UUID.

4. **Распакуйте мод** в эту подпапку.  
   - Если у вас ZIP-архив с модом, распакуйте его содержимое прямо в папку UUID.
   - Если у вас только `content.json`, переименуйте его в `content.json` (если имя другое) и положите в папку UUID. Другие файлы ресурсов тоже должны лежать здесь (в соответствующих подпапках, например `sc3d`).

5. **Добавьте папку в модлоадер**  
   - Откройте Null's Brawl.
   - Перейдите в настройки модов (значок шестерёнки).
   - Нажмите **«Добавить папку»**, выберите папку `NullsBrawlMods`.
   - Подтвердите использование и разрешите доступ.

6. **Перезапустите игру**  
   Закройте игру полностью и откройте заново. В списке модов должен появиться ваш мод. Включите его.

**Важно:** после добавления папки не нужно повторять шаг 5 для новых модов — достаточно просто создавать новые подпапки с UUID и класть туда файлы. При следующем запуске игры они появятся в списке.

**Скриншоты процесса:** https://t.me/nb_mods/1/118704

### Способ 3: Утилита Nulls TestMods (от darkmean)

Существует неофициальная утилита, которая автоматизирует процесс установки модов для тестирования.

**Как пользоваться:**
1. Скачайте и установите приложение Nulls TestMods (ищите в чате).
2. Разрешите доступ ко всем файлам.
3. В проводнике или Telegram выберите ваш `.zip` или `.json` файл и откройте через «Nulls TestMods».
4. Утилита сама создаст папку `NullsBrawlMods`, распакует мод с правильным UUID.
5. Затем в Null's Brawl добавьте папку `NullsBrawlMods` (если ещё не добавляли) и включите мод.

**Ограничения:** пока не поддерживает `.toml` файлы.

**Внимание:** это сторонняя утилита, не связанная с Null's Team. Используйте на свой страх и риск, только для своих модов.
