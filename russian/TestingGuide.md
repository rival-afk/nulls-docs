## Дополнительный гайд 6: Тестирование модов без подписи

Чтобы тестировать моды в процессе разработки, не обязательно каждый раз подписывать их. Null's Brawl предоставляет специальный APK для разработчиков, который позволяет загружать моды напрямую из ZIP, JSON или TOML файлов.

### Способ 1: Специальный APK для разработчиков

Начиная с версии 65.165 существует отдельный APK, который позволяет загружать неподписанные моды.

**Где скачать:**  
Ссылка на актуальную версию публикуется в анонсах в Telegram-канале hvh (https://t.me/hvh). Ищите сообщения с упоминанием «developer APK» или «APK для разработчиков».

**Важно:** этот APK предназначен **только для разработчиков** для тестирования собственных модов. Установка чужих неподписанных модов запрещена и может привести к блокировке аккаунта.

### Способ 2: Утилита Nulls TestMods (от darkmean)

Существует неофициальная утилита, которая автоматизирует процесс установки модов для тестирования.

**Как пользоваться:**
1. Скачайте и установите приложение Nulls TestMods (ссылка доступна в чате мододелов).
2. Предоставьте приложению доступ к файлам.
3. В проводнике или Telegram выберите ваш `.zip`, `.json` или `.toml` файл и откройте через «Nulls TestMods».
4. Утилита сама создаст папку `NullsBrawlMods` во внутреннем хранилище и распакует мод с правильным UUID.
5. В Null's Brawl добавьте папку `NullsBrawlMods` через настройки модов (шестерёнка → «Добавить папку»), если вы ещё не делали этого.
6. Включите мод в списке и перезапустите игру.

**Ограничения:** утилита автоматически генерирует UUID на основе названия мода, но для точного контроля лучше указывать UUID вручную в структуре папок.

**Внимание:** это сторонняя утилита, не связанная с Null's Team. Используйте её только для тестирования собственных модов.

### Способ 3: Ручная установка через папки (только для developer APK)

Этот метод работает **только в developer APK** из первого способа. В обычном клиенте загрузка из папок недоступна.

**Пошагово:**
1. Создайте папку для модов во внутреннем хранилище, например `NullsBrawlMods`.
2. Сгенерируйте UUID для вашего мода (см. гайд по UUID).
3. Внутри папки для модов создайте подпапку с именем, точно совпадающим с UUID.
4. Распакуйте файлы мода (`content.json`, ресурсы и т.д.) прямо в папку с UUID.
5. В developer APK откройте настройки модов (шестерёнка) → «Добавить папку» → выберите `NullsBrawlMods`.
6. Перезапустите игру — мод появится в списке.

**Важно:** после добавления папки один раз, для новых модов достаточно создавать новые подпапки с UUID — они автоматически появятся в списке при следующем запуске игры.
